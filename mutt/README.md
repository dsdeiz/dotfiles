## Mail Configuration

These are a combination of tools that makes up the whole mail client. `mutt` is the actual mail client. It's configured to make use of several tools for sending, reading, etc.

### Tools:

* [`urlview`](http://linuxcommand.org/man_pages/urlview1.html)
* [`offlineimap`](http://www.offlineimap.org/)
* [`notmuch`](https://notmuchmail.org/)
* [`w3m`](http://w3m.sourceforge.net/)
* [`pass`](https://www.passwordstore.org/)
* [`abook`](http://abook.sourceforge.net/)
* [`msmtp`](http://msmtp.sourceforge.net/)
