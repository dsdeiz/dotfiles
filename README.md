## Installation

1. Go to `$HOME`.
2. `git clone https://github.com/dsdeiz/dotfiles.git --recurse-submodules`
3. `cd dotfiles`

To install the configurations for each 'application', use `stow [application]` e.g. for zsh, `stow zsh`.

# Requirements

* [Stow][1]
* Zsh

[1]: http://www.gnu.org/software/stow/
